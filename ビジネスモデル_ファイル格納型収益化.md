# ギャル協会 × うさたにパイセン × ENTIAL × サイバーバズ
## ファイル格納型コンテンツ収益化ビジネスモデル

---

## 全体コンセプト

> **ギャル文化のコンテンツ資産をデジタルに格納し、サイバーバズの広告・PR力で収益化する**

うさたにパイセン / ギャル協会が生み出すコンテンツ（写真・動画・音声・テキスト）を
「ギャル文化アーカイブ」としてクラウドに格納。
ENTIALがその資産管理と商品化を担い、サイバーバズが企業へ販売・運用する構造。

---

## 3社（+1名）それぞれの役割

| プレイヤー | 役割 | アウトプット |
|---|---|---|
| **ギャル協会 / うさたにパイセン** | コンテンツ生産者・IP保有者 | 写真・動画・テキスト・検定コンテンツ・ギャル語録 |
| **ENTIAL** | ファイル管理・商品設計・橋渡し | コンテンツカタログ化・企業向けパッケージ設計・進行管理 |
| **サイバーバズ** | マネタイズ実行・資金提供 | 広告販売・インフルエンサーマッチング・企業PR提案 |

## ファイル格納→収益化の仕組み（コアモデル）

```
┌─────────────────────────────────────────────────────────────┐
│  STEP 1：コンテンツ格納                                       │
│  うさたにパイセン / ギャル協会                                  │

│  ├── 動画（YouTube / TikTok / リール素材）                     │

├─────────────────────────────────────────────────────────────┤
│  STEP 2：資産データベース化（ENTIALが管理）                      │
│  ├── ジャンル別整理（美容・ファッション・カルチャー・海外向け）      │
│  ├── 使用権・ライセンス条件の整備                               │
│  └── 企業向けメニュー化（タイアップ商品として設計）               │
│                   ↓ サイバーバズへ渡す                         │
├─────────────────────────────────────────────────────────────┤
│  STEP 3：サイバーバズが企業へ販売・運用                          │
│  ├── ①広告タイアップ：コンテンツに企業を乗せて配信               │

│  ├── ③インフルエンサーマッチング：うさたにパイセン案件獲得         │
│  └── ④コンテンツメディア化：アーカイブをメディアとして広告販売     │
└─────────────────────────────────────────────────────────────┘
```

## 収益化メニュー（サイバーバズが売るもの）

### A. コンテンツタイアップ広告
- 格納済みコンテンツに企業ロゴ・商品を乗せてSNS配信
- 例：コスメブランド × うさたにパイセンのメイク動画
- **単価目安：50万〜300万円 / 案件**

### B. ギャルIPライセンス
- 「ギャル協会公認」ロゴ・うさたにパイセン監修の使用権を企業に販売
- 例：ウィッグメーカー・アパレル・コスメ等が公認ロゴを使用
- **単価目安：月額30万〜100万円 / 社**

### C. インフルエンサーPRパッケージ
- うさたにパイセンのSNS投稿 × サイバーバズのPR拡散をセット販売
- 格納コンテンツをリサイクル活用することで低コスト高回転
- **単価目安：100万〜500万円 / パッケージ**

### D. ギャル文化メディア広告
- アーカイブをもとにYouTube/Webメディアを運営し、広告枠を販売
- 海外向け展開で英語コンテンツも格納→グローバル広告収益
- **単価目安：広告収益 + タイアップ枠販売**

### E. ギャル検定コンテンツ販売 ※既存事業連携
- 格納済み検定コンテンツをサイバーバズ経由で企業研修・福利厚生メニューに販売
- **単価目安：法人向け20万〜50万円 / 社**

## お金の流れ

```
企業（広告主）
    │
    │ 広告費・ライセンス料・PR費
    ▼
サイバーバズ ─── コンテンツ活動費を全額負担 ──→ ギャル協会（リスクゼロ）
    │
    │ 固定費 ＋ レベニューシェア
    ▼
  ENTIAL（プロジェクト管理・橋渡し）

【レベニューシェア比率（たたき台）】
  サイバーバズ ：ギャル協会 ＝ 6：4（要交渉、既存資料では7:3ベース）
  ENTIALは別途サイバーバズから固定費＋売上連動ボーナス
```

## ファイル格納ルール（ENTIALが整備）

| 項目 | 内容 |
|---|---|
| 格納場所 | Google Drive / Notion DB / 専用CMSのいずれか |
| タグ設計 | ジャンル・人物・使用可否・有効期限・解像度 |
| 権利管理 | 使用許可レベル（内部のみ / 企業提案用 / 一般公開） |
| 更新ルール | うさたにパイセンが週次でアップ → ENTIAL週次でカタログ更新 |
| 収益トリガー | ENTIALがサイバーバズへ月次レポートで販売可能コンテンツを共有 |
