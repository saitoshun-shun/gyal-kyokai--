# ギャル協会 経済条件提案書
**ギャル協会 × サイバーバズ × ENTIAL｜インバウンド共同事業**
*作成日：2026年4月22日（最終確定版）*

---

## 1. 基本スタンス

> **ギャル協会はIPオーナー・コンテンツクリエイターとして参加。**
> TikTok/IGのコンテンツ出演と、EC物販の元締め（ショップオーナー）の2軸で収益を受け取る。

---

## 2. 収益の受け取り方（2軸）

### 軸①｜タイアップ広告（TK・IGメディア）

企業がバズのTikTok/IGアカウントにタイアップ広告を出稿した際に発生する報酬。

| 項目 | 条件 |
|---|---|
| **報酬形態** | 件数×固定報酬 |
| **単価** | **1件あたり25万円** |
| **月最低保証** | **10万円**（受注ゼロ月も10万円をお支払い） |
| **支払いタイミング** | 月末締め翌月末払い |

**Year1受取シミュレーション（タイアップのみ）：**

| シナリオ | タイアップ件数 | 受取額 |
|---|---|---|
| ①大成功 | 46件 | **1,170万円**（最低保証含む） |
| ②及第点 | 31件 | **800万円**（最低保証含む） |
| ③撤退 | 6件 | **230万円**（最低保証補填含む） |

---

### 軸②｜TikTok Shop EC物販（ショップオーナー）

ギャル協会がTikTok Shopのショップオーナーとして、商品の企画・製造・在庫・発送・CSを担う。バズはアフィリエイターとして商品リンクを設置し、売上向上に貢献する。

**ギャル協会の業務スコープ：**

| 業務 | 詳細 |
|---|---|
| 商品企画・デザイン監修 | コスメ・アパレル・キャラグッズ等 |
| OEMメーカー選定・発注 | 小ロット（50〜100個）から開始 |
| TikTok Seller Center登録・出品 | ギャル協会名義のショップ |
| 在庫管理・入出庫 | 自社またはFCで管理 |
| 発送・物流管理 | 国内外への配送対応 |
| CS（問い合わせ・返品対応） | 日英対応が望ましい |

**収益の構造（EC売上100%に対して）：**

```
EC実売上（100%）
  − 仕入原価（45%）
  − TikTok Shop手数料（7%）
  − 物流費（4%）
  − 在庫損失（1%）
  ─────────────────────────
  EC純利益（43%）
  − バズへのアフィリエイトコミッション（20%）
  ─────────────────────────
  ギャル協会の手残り：EC売上の約23%
```

**Year1受取シミュレーション（EC分）：**

| シナリオ | EC売上 | 手残り（約23%） |
|---|---|---|
| ①大成功 | 8,100万円 | **約1,863万円** |
| ②及第点 | 4,302万円 | **約990万円** |
| ③撤退 | 571万円 | **約131万円** |

## 4. ギャル協会が担うこと・担わないこと

| 担当 | 内容 |
|---|---|
| ◎ | TK・IGコンテンツへの出演・企画・監修 |
| ◎ | EC商品企画・デザイン・OEM発注 |
| ◎ | TikTok Shop運営（出品・在庫・発送・CS） |
| ◎ | 体験ツアーの当日出演・実施（別途協議） |
| − | TK・IGメディア運営・投稿管理（バズが担当） |
| − | タイアップ受注・管理（バズが担当） |
| − | OTA掲載・予約・法人営業（ENTIALが担当） |

> **「コンテンツ × EC物販の二刀流」として、IPの世界観で商品も売る体制。**
> メディア運営や営業の中身には口出しせず、クリエイティブとオペに集中していただく。

## 5. スケールアップ時の条件見直し

| フェーズ | 条件 | 見直し内容 |
|---|---|---|
| M6〜 | ECが月次300万超 | 仕入原価の交渉（45%→35%目標）でギャル協会の手残り増加 |
| Year2〜 | EC安定後 | Shopify経由の越境EC展開を協議（SKU拡張） |
| Year2〜 | 事業が①②水準で安定 | タイアップ単価・件数の上方改定を協議 |

---

## 6. 事業が撤退になった場合

- M9末のKPIゲートで撤退判断となった場合、翌月より契約を終了
- 撤退決定以降の月最低保証は打ち切り
- EC在庫の処分方法・残余利益分配は別途協議
- 既発生分の報酬は全額お支払い
