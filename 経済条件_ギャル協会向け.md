## 2. 収益の受け取り方（2軸）

### 軸①｜タイアップ広告（TK・IGメディア）

企業がバズのTikTok/IGアカウントにタイアップ広告を出稿した際に発生する報酬。

| 項目 | 条件 |
|---|---|
| **報酬形態** | 件数×固定報酬 |
| **単価** | **1件あたり25万円** |
| **月最低保証** | **10万円**（受注ゼロ月も10万円をお支払い） |
| **支払いタイミング** | 月末締め翌月末払い |

**なぜ件数×固定か：**
- タイアップ単価（150万円）の変動に関わらず、安定した報酬を受け取れる
- 受注が増えるほど報酬も増える設計（インセンティブが一致）

**Year1受取シミュレーション（タイアップのみ）：**

| シナリオ | タイアップ件数 | 受取額 |
|---|---|---|
| ①大成功 | 46件 | **1,150万円** |
| ②及第点 | 31件 | **775万円** |
| ③撤退 | 6件 | **220万円**（最低保証補填含む） |

---

### 軸②｜TikTok Shop EC物販（グッズ）

ギャル協会公認グッズ（コスメ・アパレル・キャラグッズ等）の売上に対するレベニューシェア。

| 項目 | 条件 |
|---|---|
| **報酬形態** | EC売上×レベニューシェア |
| **料率** | **EC売上の15%** |
| **支払いタイミング** | 月末締め翌月末払い（TikTok Shop入金確認後） |
| **対象** | ギャル協会監修・公認グッズの全売上 |

**Year1受取シミュレーション（EC分）：**

| シナリオ | EC売上 | 受取額（15%） |
|---|---|---|
| ①大成功 | 8,100万円 | **1,215万円** |
| ②及第点 | 4,302万円 | **645万円** |
| ③撤退 | 571万円 | **86万円** |

## 4. ギャル協会が担うこと（スコープ）

| 担当 | 内容 |
|---|---|
| ◎ | TK・IGコンテンツへの出演・企画・監修 |
| ◎ | グッズの企画・デザイン監修 |
| ◎ | 体験ツアーの出演・実施（別途協議） |
| − | メディア運営・投稿管理（バズが担当） |
| − | EC在庫・物流・CS（バズ＋ENTIALが担当） |
| − | OTA掲載・営業（ENTIALが担当） |

> **「中の運営には口出ししない。コンテンツと世界観の提供に集中する」を明確に約束します。**

## 5. 事業が撤退になった場合

- M9末のKPIゲートで撤退判断となった場合、翌月より契約を終了
- 撤退決定以降の月最低保証は打ち切り
- 既発生分の報酬は全額お支払い
