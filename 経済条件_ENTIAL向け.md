# ENTIAL 経済条件提案書
**ギャル協会 × サイバーバズ × ENTIAL｜インバウンド共同事業**
*作成日：2026年4月22日（最終確定版）*

---

## 1. 基本スタンス

> **ENTIALはこの事業の「3つの顔」を担う。**
> ①バズ社内PMとして3社の進行を管理、②タイアップ営業をサポート、③体験事業の元締めとして売上を計上。
> 特に③体験事業がENTIALの最大の稼ぎどころとなる。

---

## 2. 収益の受け取り方（2軸）

### 軸①｜タイアップ成功報酬

バズが受注したタイアップ広告売上に対する成功報酬。

| 項目 | 条件 |
|---|---|
| **報酬形態** | 完全成功報酬 |
| **料率** | **タイアップ売上の5%** |
| **固定費** | なし |
| **支払いタイミング** | 受注確定後、翌月末払い |

**Year1受取シミュレーション（タイアップ成功報酬）：**

| シナリオ | タイアップ売上 | ENTIAL受取（5%） |
|---|---|---|
| ①大成功 | 6,900万円 | **345万円** |
| ②及第点 | 4,500万円 | **225万円** |
| ③撤退 | 600万円 | **30万円** |

---

### 軸②｜体験事業元締め（最大の稼ぎどころ）

ENTIALがインバウンド向け体験ツアー（ギャル変身・渋谷ナイトツアー等）のOTA掲載から予約・CS・売上計上まで全てを担う元締め役。

**ENTIALの元締め業務スコープ：**

| 業務 | 詳細 |
|---|---|
| 体験メニュー設計・価格設定 | 単価1.5万〜50万円のライン整備 |
| OTA掲載・運用 | Rakuten Travel / Klook / GetYourGuide / Viator |
| 多言語問い合わせ対応 | 英・中・韓のインバウンドCS |
| 予約管理・リマインド | 当日調整・通訳手配含む |
| 体験売上の計上・財務管理 | OTA入金管理・請求処理 |
| 法人営業（B2B） | ホテル・旅行代理店・企業インセンティブ |
| プレスツアー企画 | インフルエンサー・メディア向け |

**体験売上の分配構造：**

```
体験売上（100%）
  − ギャル協会への出演費（30〜40%）
  − バズへのメディア利用料（15〜20%）
  ─────────────────────────
  ENTIAL手残り：体験売上の40〜55%
```

**Year1受取シミュレーション（体験事業）：**

| 想定体験売上 | ENTIAL手残り（約40%） | 月平均 |
|---|---|---|
| 年間1,800万円 | **約720万円** | **約60万円** |
| 年間2,400万円 | **約960万円** | **約80万円** |

**体験メニュー別単価（ENTIALが設定）：**

| メニュー | 単価 | 所要時間 |
|---|---|---|
| ギャル変身＋渋谷フォトシュート | 2.5〜4万円/人 | 3〜4時間 |
| 渋谷夜遊びツアー | 5〜8万円/人 | 4〜5時間 |
| ギャル文化ワークショップ | 1.5〜2.5万円/人 | 2時間 |
| 企業・インフルエンサー貸切 | 20〜50万円/団体 | 半日〜1日 |

## 3. Year1 合計受取シミュレーション（ENTIAL）

| シナリオ | TIE成功報酬（5%） | 体験元締め利益 | **年間合計** |
|---|---|---|---|
| **①大成功** | 345万円 | 約720万円 | **約1,065万円（月89万円）** |
| **②及第点** | 225万円 | 約360万円 | **約585万円（月49万円）** |
| **③撤退** | 30万円 | 約90万円 | **約120万円（月10万円）** |

> **①大成功時は月89万円。バズ社内PM常駐のコスト対比として十分な水準を目指す。**
> ③撤退シナリオではENTIALの収益は月10万円と低水準になる。KPIゲートを事前合意し、M9末で撤退判断する仕組みで担保する。

## 4. ENTIALが担うこと・担わないこと

### 担うこと

| 領域 | 業務 |
|---|---|
| **体験事業（元締め）** | OTA掲載・予約・CS・売上計上・法人営業・財務管理 |
| **タイアップ営業サポート** | 候補企業リスト・提案資料・商談アポ支援 |
| **PM常駐** | 3社進行管理・KPI監視・月次PLレビュー・議事録 |
| **ギャル協会との調整** | スケジュール・コンテンツ供給・当日オペ |

### 担わないこと（バズ・ギャル協会の領域）

| 領域 | 担当 |
|---|---|
| TK・IGの投稿運用・広告配信 | バズ |
| ECの商品企画・在庫・発送・CS | ギャル協会 |
| WESELLの設定・アフィリエイト運用 | バズ |
| コンテンツ編集・字幕制作 | バズ |
| 制作費・メディア原価の管理 | バズ |

## 5. 撤退・契約終了条件

| タイミング | 条件 | 対応 |
|---|---|---|
| M6末 | タイアップ累計3社未満 or EC累計200件未満 | 双方でKPIレビュー・施策見直し協議 |
| M9末 | 月次タイアップ2社未満 or EC月次売上300万未満 | **撤退決定・契約終了の協議開始** |
| M10〜M12 | 撤退決定後 | 新規受注停止・既存案件クロージングのみ |

## 6. 将来的な報酬見直し（Year2以降）

| 条件 | 見直し内容 |
|---|---|
| 体験売上が月200万を超えて安定 | ENTIALの取り分を段階的に引き上げ（55%上限を協議） |
| 事業が②及第点以上で安定 | タイアップ成功報酬率の見直し（5%→上方改定を協議） |
| 海外展開・越境EC開始 | 新規ラインへの追加インセンティブを設定 |
