## ギャル協会 × サイバーバズ × ENTIAL

> **更新日：2026年4月22日**

---

## 1. 事業の全体像（3軸）

| 軸 | プラットフォーム | 収益の主体 | バズの収益形態 |
|---|---|---|---|
| **①タイアップ広告** | TikTok・Instagram | バズ（売上計上） | 広告売上（粗利73%） |
| **②EC物販** | TikTok Shopアフィリエイト | ギャル協会（ショップオーナー） | コミッション（EC実売上×20%） |
| **③IG体験コンテンツ** | OTA（Rakuten Travel等） | ENTIAL（元締め） | メディア利用料（体験売上×15〜20%） |

**基本方針：**
- バズのPLに乗るのは**タイアップ売上のみ**（シンプル・高粗利・管理基盤不要）
- EC・体験は「場所・メディアを提供して手数料をもらう」立場

## 2. 3社の基本スタンス

```
┌─────────────────────────────────────────────────────────────┐
│ ギャル協会 / うさたにパイセン                                │
│   ▸ コンテンツ・IP・世界観創出（タイアップ出演・監修）       │
│   ▸ EC物販：商品企画→製造（OEM）→在庫→発送→CS             │
│     ＝ TikTok Shopのショップオーナー                         │
│   ▸ 体験ツアー：当日出演・実施                              │
│   ※ メディア運営・タイアップ受注の中身には口出しない        │
├─────────────────────────────────────────────────────────────┤
│ サイバーバズ                                                 │
│   ▸ TK・IGメディア運営（投稿・広告配信）                    │
│   ▸ タイアップ広告の受注・管理（売上計上）                  │
│   ▸ TikTok Shopアフィリエイター登録（コミッション受取）      │
│   ▸ WESELLを活用したアフィリエイト商品リンク設置            │
│   ▸ 制作費・メディア原価の管理                              │
├─────────────────────────────────────────────────────────────┤
│ ENTIAL                                                        │
│   ▸ 体験事業の元締め（OTA掲載・予約・CS・法人営業・売上計上）│
│   ▸ バズ社内PMとして常駐（3社進行・KPI管理）                │
│   ▸ タイアップ営業サポート（成功報酬TIE×5%）                │
│   ▸ EC運用は担わない（ギャル協会が主体）                    │
└─────────────────────────────────────────────────────────────┘
```

## 3. 収益の流れ（全体）

```
【①タイアップ広告】
企業 →（広告費）→ バズ（売上計上・メディア運営）
                      ↓ ギャル協会に出演・監修料（件数×25万）
                      ↓ ENTIALに成功報酬（TIE×5%）

【②EC物販】
消費者 → TikTok Shop → ギャル協会（売上計上・在庫・発送・CS）
              ↑               ↓ バズにアフィリエイトコミッション（EC×20%）
         バズのWESELL動画で
         商品リンク設置

【③体験コンテンツ】
訪日外国人 → OTA → ENTIAL（売上計上・予約・CS）
                        ↓ ギャル協会に出演費（体験売上の30〜40%）
                        ↓ バズにメディア利用料（体験売上の15〜20%）
```

## 5. 作業WF②｜EC物販（TikTok Shopアフィリエイト）

### バズとギャル協会の役割分担

```
ギャル協会（ショップオーナー）     バズ（アフィリエイター）
─────────────────────────          ─────────────────────
商品企画・デザイン監修              TikTok Shopアフィリエイター登録
OEMメーカー選定・発注              動画内に商品リンク設置（WESELL活用）
在庫管理・入出庫                   切り抜き・UGC拡散施策
TikTok Shop出品・商品ページ         コミッション受取（EC実売上×20%）
発送・物流管理
CS（問い合わせ・返品対応）
```

### 詳細ステップ

#### STEP 1｜ショップ・商品準備（M1〜3）
| 項目 | 担当 | 備考 |
|---|---|---|
| TikTok Seller Center登録 | ギャル協会 | ギャル協会名義のショップ |
| 商品OEM発注（小ロット50〜100個） | ギャル協会 | コスメ・アパレルから開始 |
| 商品ページ作成（日英対応） | ギャル協会＋バズ（英語監修） | |
| バズアカウントのアフィリエイター申請 | バズ | ギャル協会ショップに申請 |

#### STEP 2｜アフィリエイト配信（M3〜）
| 項目 | 担当 |
|---|---|
| 商品紹介動画制作・投稿 | ギャル協会（出演）＋バズ（編集・投稿） |
| WESELLで商品リンク設置 | バズ |
| 切り抜き機能でUGC拡散 | バズ |
| コミッション入金確認 | バズ |

#### STEP 3｜スケール（M6〜）
- 売れたSKUのみロット増加（ギャル協会判断）
- 仕入原価交渉（45%→35%目標）
- Shopify経由の越境EC展開を検討（Year2〜）

## 6. 作業WF③｜IG体験コンテンツ（ENTIAL元締め）

### 全体フロー

```
【メニュー設計】→【OTA掲載】→【集客・予約】→【体験実施】→【素材活用・継続】
  ENTIAL主導      ENTIAL主導   ENTIAL主導    ギャル協会    バズ+ENTIAL
```

### 体験メニュー（ENTIALが設計・管理）

| メニュー | 単価 | 所要時間 |
|---|---|---|
| ギャル変身＋渋谷フォトシュート | 2.5〜4万円/人 | 3〜4時間 |
| 渋谷夜遊びツアー | 5〜8万円/人 | 4〜5時間 |
| ギャル文化ワークショップ | 1.5〜2.5万円/人 | 2時間 |
| 企業・インフルエンサー貸切 | 20〜50万円/団体 | 半日〜1日 |

### ENTIALが担う具体的な業務

#### A. OTA掲載・管理
- Rakuten Travel Experiences / Klook / GetYourGuide / Viatorへの掲載申請
- 英語・中国語・韓国語対応の商品説明文作成
- 料金設定・予約枠管理・レビュー返信

#### B. 予約・CS対応
- OTA経由の予約確認・リマインド送信
- 多言語問い合わせ対応（英・中・韓）
- キャンセル・日程変更処理

#### C. 当日オペレーション調整
- ギャル協会との当日スケジュール・台本共有
- 通訳・アテンド手配（必要時）
- 体験後の写真・動画素材をバズに納品

#### D. 法人営業
- ホテル・旅行代理店への体験パッケージ提案
- 企業インセンティブツアーへの売り込み
- インフルエンサー・メディアのプレスツアー企画

#### E. 財務管理
- OTA入金管理・請求処理
- ギャル協会への出演費支払い（体験売上の30〜40%）
- バズへのメディア利用料支払い（体験売上の15〜20%）
