# 事業計画書
## ギャル×インバウンド コンテンツビジネス
### ギャル協会 × サイバーバズ × ENTIAL

---

## 1. 事業概要

### ビジョン
> 「ギャル文化」を日本最強のインバウンドコンテンツIPとして世界に届ける

### 一言で言うと
訪日外国人が求める「本物の日本サブカルチャー体験」を、
ギャル協会／うさたにパイセンのIPをコアに、
TikTokショートドラマメディア × リアル体験 × 企業タイアップで収益化する事業。

---

## 2. 市場背景

- 訪日外国人数は2024年に過去最高を更新、インバウンド消費は拡大継続
- 海外での「ギャル文化」認知がSNSを中心に急上昇（Y2Kブーム・日本サブカル人気）
- 観光庁・自治体がインバウンド向けコンテンツに積極的に予算を投下中
- **ギャル×インバウンドを商品化している競合は現時点でゼロ**

## 4. 事業モデル

### コアメディア：TikTokショートドラマアカウント（サイバーバズ名義）

| 項目 | 内容 |
|---|---|
| プラットフォーム | TikTok（メイン）、Instagram Reels（流用） |
| コンテンツ形式 | ショートドラマ（60〜180秒） |
| テーマ | ギャルが案内する日本・インバウンド体験・渋谷カルチャー |
| 更新頻度 | 週2〜3本 |
| 言語 | 日本語＋英語字幕（海外向け） |
| 運営 | サイバーバズ（制作費負担） |
| 出演・監修 | ギャル協会 / うさたにパイセン |

### 収益の流れ

```
TikTokメディアが育つ
        ↓
インバウンド関連企業・自治体への営業ネタになる
        ↓
ENTIALが営業（＋営業代理店を活用）
        ↓
タイアップ・協賛・ライセンス契約を受注
        ↓
サイバーバズに入金
        ↓
ギャル協会へレベニューシェア、ENTIALへ営業手数料
```

## 6. 営業戦略

### ターゲット優先順位

| 優先度 | ターゲット | 理由 |
|---|---|---|
| **高** | ホテル・旅行代理店 | 意思決定が速く、予算もある。インバウンド需要で今がピーク |
| **高** | 免税店・百貨店 | インバウンド消費に直接つながる。単価が高い |
| **中** | 地方自治体・観光協会 | 予算は大きいが意思決定が遅い。補助金活用で組みやすい |
| **中** | OTA・体験予約サービス | 拡散力が大きいが単価は低め。メディア成長とセットで攻める |
| **後** | 海外メディア・プラットフォーム | TikTokが育ってから本格攻略 |

### ENTIALの動き方
1. **まず営業代理店を1〜2社開拓**（インバウンド・観光系に強い代理店）
2. 代理店経由でホテル・旅行代理店・百貨店へ提案
3. 自治体案件は補助金スキームとセットで直接提案

## 7. お金の流れ

```
企業・自治体（広告主）
    │
    │ タイアップ費・協賛費・ライセンス料
    ▼
ENTIAL（受注・請求窓口）
    │
    │ 売上全額をサイバーバズへ
    ▼
サイバーバズ
    ├─ ギャル協会へ：出演費・コンテンツ費 ＋ レベニューシェア
    │   （サイバーバズ：ギャル協会 ＝ 6:4〜7:3、要交渉）
    └─ ENTIALへ：営業手数料（固定費 ＋ 売上連動インセンティブ）

※ コンテンツ制作費・TikTok運営費はサイバーバズ負担
※ ギャル協会はリスクゼロで参加できる
```

## 8. ロードマップ

### Phase 1｜仕込み期（0〜3ヶ月）
- [ ] 3社間の契約・役割・収益分配を合意
- [ ] TikTokアカウント開設・初期コンテンツ10本制作
- [ ] ENTIALが営業代理店1〜2社を開拓
- [ ] ホテル・旅行代理店への初回提案開始

### Phase 2｜初収益期（3〜6ヶ月）
- [ ] タイアップ案件 初回受注（Package 2か3から）
- [ ] TikTokフォロワー 1万人達成
- [ ] 自治体案件 1件提案・交渉開始

### Phase 3｜スケール期（6〜12ヶ月）
- [ ] 月次タイアップ案件を安定受注
- [ ] TikTok英語字幕コンテンツで海外フォロワー獲得
- [ ] ギャル協会IPの海外ライセンス販売開始
- [ ] 神7組成・複数ギャルで案件を並走できる体制構築
